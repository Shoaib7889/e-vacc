## Features

- Bootstrap 4+ & SCSS
- Responsive layout
- React Chart.js
- Simple login / logout
- Error page
- Styled Bootstrap components like buttons, modals, etc

## Instalation

1. Clone repository

2. Get in the project folder

```shell
cd e-vacc
```

3. Install dependencies via npm or yarn

```shell
npm install
```

```shell
npm start
```
